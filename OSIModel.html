Presentation Layer Details

Function: Data formatting, encryption/decryption, compression

Data Unit: Data

Services: Character encoding, data compression, encryption

SSL/TLS JPEG GIF ASCII EBCDIC MPEG
5
Session Layer
Session management and dialog control

Session Layer Details

Function: Establishes, manages, and terminates sessions

Data Unit: Data

Services: Dialog control, session checkpointing, recovery

NetBIOS RPC SQL Session LDAP
4
Transport Layer
End-to-end communication and reliability

Transport Layer Details

Function: Reliable end-to-end communication, flow control, error recovery

Data Unit: Segments (TCP) / Datagrams (UDP)

Services: Port addressing, segmentation, multiplexing

TCP UDP SPX
3
Network Layer
Routing between different networks

Network Layer Details

Function: Packet routing, logical addressing, path determination

Data Unit: Packets

Devices: Routers, Layer 3 switches

IP ICMP OSPF BGP RIP

Data Link Layer Details

Data Unit: Frames

Devices: Switches, Bridges, NICs

Ethernet Token Ring PPP WiFi (802.11) ARP
